Two Sum

Thinking and Analysis:

Solution1: Brute force, O(n^2)

Solution2: Sort - O(nlogn)) and then use two pointers to let them move from head and tail to middle - O(n).

Solution3: Hash. Using a map to store the vector’s value and let them equal to their index. NOTICE: The map must not be initialized at the beginning. For example [4, 2, 6] and target is 8, it would be error and return [1,1].(should be [2,3]) The proper way is traversing the map to detect answer and adding element to map in the same time.(Avoid same + same = target which same is the only one element).

Map C++: find() would return map.end() if the element is not found.

Runtime: 20 ms (Leetcode test)
