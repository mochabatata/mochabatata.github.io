Palindrome Partitioning

Thinking and Analysis:

Use dfs to traverse the array from every start charactors.
All path -> vector> to store result, and use vector<> path to store every suitable substring -> start index is the key to extend the status -> no need to judge repetition -> convergence when all string is traversed -> path[].pop_back is the way to recover
T: O(n^2) S: O(1)
Runtime: 16 ms (Leetcode test)
