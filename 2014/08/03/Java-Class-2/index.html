Part 3 Variables

分类:

1.primitive主数据类型：boolean char | byte short int long | float double
2.引用

对象变量：

实际上没有对象变量存在，只有引用（reference）到对象的变量，它不是对象的容器，类似指针，地址。
举例：

d相当于遥控器，”.”相当于遥控器上的按钮。

对象的声明、创建和赋值：

Dog d ——要求jvm分配空间给应用变量，命名为d，变量永远为 Dog类型
new Dog() ——要求jam分配堆空间给新建的dog对象
= —— 把新的Dog赋值给d这个引用变量

Part 4 Method

类的内容：

对象知道什么和执行什么！

传参：

实参（argument） 形参（parameter）返回值

封装：

get：返回实例变量的值
set：取用一个参数来设定实例变量的值
暴露：可利用原点运算符存取 dog.age = 2;
需要创建setter这个方法给所有的变量值，强制通过setter来设定变量
setter中可以加类似的过滤器
变量变为私有，getter和setter设为共有。

Part 5 Programming

高层设计：

需要类和方法，切记：以OO的方式思考，专注于出现的事物而不是过程

开发类：

  1. 找出类应该做的事情

  5. 实现类
